## URL Shortener

### This app lets you shrink your links and manage them.

#### Warning: Inactive users are deleted every start of month.
```
(Inactive = last link shortened > 30 days)
```

### Features
- Multiple log in methods
- Top tier security
- Links management

### Tech stack
- Next.js
- Auth.js
- Prisma
- JWE Tokens
- bcrypt
- React Hook Form
- Zod
- Tailwind
